# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Real-time progress monitoring with progress bars
- Support for HTTP/2 and HTTP/3 protocols
- Custom assertion support for response validation
- Plugin system for extending functionality
- Support for GraphQL requests
- Distributed testing capabilities

### Changed
- Improved memory usage for large CSV files
- Enhanced error reporting with more context
- Optimized connection pooling

## [0.1.0] - 2024-01-01

### Added
- Initial release of Resty-Stress-Tester
- Basic HTTP methods support (GET, POST, PUT, DELETE, PATCH, HEAD, OPTIONS)
- CSV parameterization with template support
- Multiple report formats (console, JSON, HTML)
- Configurable concurrency and request limits
- Duration-based and count-based testing modes
- Docker support
- Comprehensive test suite
- CI/CD pipeline with GitHub Actions
- Detailed documentation and examples

### Features
- High-performance stress testing engine
- Real-time statistics and metrics
- Error classification and analysis
- Flexible configuration via flags and files
- Keep-alive connection support
- TLS configuration options
- Verbose logging and progress monitoring
